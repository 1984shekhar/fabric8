fabric8: an open source integration platform (iPaaS)
================================

Welcome to fabric8, the open source integration platform for running all your open source integration technologies, like Apache ActiveMQ, Camel, CXF and Karaf in the cloud.


Get Started
--------

* Getting Started Guide for how to run fabric8 on your local computer:
   https://access.redhat.com/documentation/en/red-hat-jboss-fuse/6.3/single/fabric-guide#GetStart

* Run fabric8 on OpenShift for using fabric8 on the open hybrid cloud:
   https://www.openshift.com/quickstarts/jboss-fuse-61

   More details here:
   https://github.com/jboss-fuse/fuse-openshift-cartridge/blob/master/README.md

* Run fabric8 on docker:
   https://github.com/fabric8io/fabric8-docker#try-it-out
